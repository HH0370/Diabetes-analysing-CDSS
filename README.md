# Diabetes Analyzing Clinical Decision Support System (CDSS)

This project is a simple web-based Clinical Decision Support System for diabetes analysis. It uses Python Flask for the backend and HTML/CSS/JS for the frontend.

## Features
- Flask backend API for diabetes risk analysis
- Simple web frontend for user input and results display
- Basic folder structure for static and template files

## How to Run
1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Start the Flask server:
   ```
   python app.py
   ```
3. Open your browser and go to `http://127.0.0.1:5000/`

## Project Structure
- `app.py` - Flask backend
- `templates/index.html` - Main web page
- `static/style.css` - Stylesheet
- `static/script.js` - Frontend logic
- `requirements.txt` - Python dependencies

## Note
- The diabetes analysis logic is a simple rule-based example. Replace it with a real model or logic as needed.
